# JARVIS---IA-ASSISTENTE-2.0
JARVIS é uma IA que ve sua tela e escuta sua voz, programada para ser seu assitente durante o dia dia no trabalho.
-------------------------------------------------------------------------------------------------------------------
Tem automaçao - Abre apps, faz pesquisa, ajuda no que pedir, desliga o computador, reinicia e muito mais.
-----------------------------------------------------------------------------------------------------------
Criado 100% em python 
---------------------------------
Usa API Groq (versao gratuita) 
---------------------------------
Inspirada no J.A.R.V.I.S de Homem de ferro, 
fiz ela para me ajudar a estudar e automatizar meus trabalhos do dia a dia, 
me ajudando em a corrigir erros e melhorando meu desenvolvimento e estudos, 
ele é como um professor
